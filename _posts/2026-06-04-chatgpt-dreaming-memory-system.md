OpenAI has deployed a new memory architecture for ChatGPT called "Dreaming," designed to keep user preferences and context fresh across conversations. The system moves beyond the earlier explicit memory feature toward more continuous, passive context retention. The capability raises privacy questions about what data persists, for how long, and whether users can inspect or purge the inferred memory state—details OpenAI has not yet fully documented publicly.
