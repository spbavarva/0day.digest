OpenAI is rolling Codex, its AI coding agent capable of writing code and automating tasks, into
the ChatGPT mobile app on iOS and Android as a preview. The move is a competitive response to
the growing popularity of Anthropic's Claude Code in the agentic developer tooling space.

The mobile integration gives developers access to Codex's code-writing and workflow automation
capabilities outside the desktop environment. OpenAI has been accelerating Codex development and
distribution following Claude Code's strong market reception.
