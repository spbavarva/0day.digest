Google and Mozilla shipped updates patching multiple critical and
high-severity memory safety bugs in Chrome and Firefox. The flaws could
potentially be leveraged for remote code execution if exploited. No detail
was provided on active exploitation in the wild. Users and IT teams should
update both browsers to the latest stable versions, since memory safety bugs
in browser engines remain a recurring vector for drive-by RCE chains.
