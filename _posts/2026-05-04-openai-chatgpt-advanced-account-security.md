OpenAI rolled out an Advanced Account Security feature for ChatGPT that includes stronger login methods, more secure account recovery flows, shorter session lifetimes, and an option to exclude account data from model training. The feature is aimed at users and organizations concerned about account compromise and data privacy.

Shorter sessions reduce the blast radius of stolen session tokens — a common attack vector against SaaS platforms. Enterprise security teams managing ChatGPT Enterprise or Teams accounts should evaluate the new security options, enforce the stronger authentication methods where available, and review their existing policies around AI tool usage and data classification.
