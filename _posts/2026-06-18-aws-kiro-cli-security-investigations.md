AWS released Kiro CLI, a command-line tool aimed at speeding up security
investigations in AWS environments. It targets the time-consuming, manual
work analysts currently do recalling AWS CLI syntax and correlating findings
across services like Amazon GuardDuty and AWS CloudTrail. AWS positions it
as a way to cut investigation time during active security events. Teams
running AWS workloads may want to evaluate Kiro CLI as part of their
incident response tooling.
