An active campaign is using WhatsApp Desktop and WhatsApp Web direct messages
to distribute malicious VBScript files disguised as business documents.
Victims who run the script get a legitimate ManageEngine remote monitoring
and management (RMM) tool installed, giving attackers remote access while
blending in with normal admin tooling. Kaspersky has tracked the campaign
across Malaysia, Brazil, India, Mexico, Singapore, the U.K., Spain, Taiwan,
and Australia. Treat unsolicited WhatsApp document attachments — especially
`.vbs` files — as a remote-access risk, not just a phishing nuisance.
