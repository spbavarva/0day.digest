QiAnXin's XLab identified a new malware family, AryStinger, that has infected at least 4,300 legacy home routers, with the count still rising. Unlike typical router botnets built for DDoS, AryStinger turns infected devices into a distributed reconnaissance and proxy network used to scout targets and relay traffic before an attack is launched. Router owners should apply firmware updates and retire end-of-life hardware that no longer receives security patches.
