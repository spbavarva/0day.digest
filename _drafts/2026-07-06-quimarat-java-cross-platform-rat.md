LevelBlue researchers flagged QuimaRAT, a new Java-based remote access
trojan capable of targeting Windows, Linux, and macOS from a single
codebase. The malware is sold as malware-as-a-service, with pricing tiers
from $150 for one month up to $1,200 for lifetime access. Cross-platform,
subscription-priced RATs lower the bar for less technical attackers to run
persistent remote access campaigns across mixed-OS environments. Defenders
should watch for Java-based RAT indicators alongside more common
native-binary RAT signatures.
