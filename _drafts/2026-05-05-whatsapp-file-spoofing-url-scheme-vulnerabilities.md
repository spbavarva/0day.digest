Meta disclosed two WhatsApp vulnerabilities—a file spoofing flaw and an arbitrary URL scheme issue—surfaced through its bug bounty program. Both were patched in updates released earlier this year. No active exploitation has been reported.

Users on current WhatsApp versions are not at risk. The file spoofing flaw could allow attackers to disguise malicious files as benign types; the URL scheme issue could redirect users to attacker-controlled destinations via crafted messages.
