Trend Micro, Tanium, ESET, and Tenable each shipped patches this week for
critical and high-severity vulnerabilities in their security products.
Specific CVE identifiers, affected product versions, and exploitation
details were not included in initial reporting. Because these are security
tools with privileged access to endpoints and networks, unpatched instances
represent an elevated risk if exploited. Administrators running any of these
products should check vendor advisories and apply the latest updates as soon
as possible.
