Unit 42 researchers identified a new forensic artifact in macOS Tahoe 26 that records a history of user menu selections across the operating system.
The artifact can help investigators reconstruct a timeline of user actions during incident response or forensic analysis.
Defenders performing macOS forensics should be aware this data source exists and consider adding it to collection workflows.
No exploitation or security weakness was described — this is a forensic data-source discovery, not a vulnerability.
