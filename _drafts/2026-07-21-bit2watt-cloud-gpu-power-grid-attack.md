Researchers at Zhejiang University describe Bit2Watt, an attack accepted to
CHES 2026 (the IACR's hardware-security conference), in which a cloud
tenant using nothing but ordinary GPU access can rapidly swing a data
center's power draw up and down — fast enough to threaten the power grid
the data center runs on. No exploit or break-in is required; the technique
relies purely on legitimate GPU workload patterns.
