Claims have circulated that LinkedIn is conducting large-scale browser surveillance through a
browser extension — labeled "BrowserGate" online. Independent security researchers analyzing
the extension found conflicting evidence: some behaviors are consistent with the surveillance
claims; others appear to be standard telemetry and analytics. The situation remains unresolved.

Microsoft, which owns LinkedIn, has not issued a formal technical response to the specific
allegations. Until the scope of data collection is clearly documented, users with elevated
privacy requirements should review all permissions granted to the LinkedIn browser extension
and consider disabling it. Organizations running LinkedIn extensions across managed endpoints
should evaluate whether the data collection behaviors are compatible with their acceptable use
and data handling policies.
