Security researcher "Chaotic Eclipse" (Nightmare-Eclipse) released a
proof-of-concept exploit called LegacyHive, described as a Windows User
Profile Service (ProfSvc) arbitrary hive load elevation-of-privilege
vulnerability. ProfSvc is a core Windows component that manages user accounts
and environments. The PoC was published hours after this month's Patch
Tuesday.
