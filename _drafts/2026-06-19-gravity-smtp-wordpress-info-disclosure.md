Threat actors are actively exploiting an unauthenticated information
disclosure vulnerability in the Gravity SMTP WordPress plugin, which is
installed on roughly 100,000 sites. The flaw lets attackers access
sensitive information without authenticating. No CVE identifier or
patch status was given in the available summary. Site operators running
Gravity SMTP should check for an update and review logs for suspicious
access.
