Cisco released the Antares models, a set of open-weight AI models built to
pinpoint known vulnerabilities in codebases. Cisco positions Antares as
faster and significantly cheaper to run than larger general-purpose AI
models while targeting the same source-code vulnerability detection use
case. The models are open-weight, making them available for teams to
self-host rather than relying on a hosted API.
