Microsoft says Windows users should expect a higher volume of security
updates going forward as the company increasingly uses AI to identify
vulnerabilities in its codebase earlier in development. Microsoft frames
this as more issues caught before release rather than a sign of degrading
code quality. The company also noted that attackers, including amateurs,
are increasingly using AI to find and exploit vulnerabilities themselves.
