Meta's WhatsApp has disclosed two vulnerabilities: a file spoofing flaw that could allow attackers to disguise malicious files as benign ones, and an arbitrary URL scheme vulnerability. Both were reported through Meta's bug bounty program and patched in WhatsApp updates released earlier in 2026. While patches are available, the disclosure serves as a reminder that chat platform clients remain an active attack surface for file-based social engineering. Users should ensure WhatsApp is updated to the latest version across all platforms.
