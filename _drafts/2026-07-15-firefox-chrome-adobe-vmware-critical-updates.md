Mozilla patched two critical Firefox flaws: CVE-2026-15718, an invalid
pointer bug in the WebAssembly component, and CVE-2026-15719, a site
isolation bypass in DOM Navigation. Exploit code for these is already public,
though Mozilla says it isn't yet aware of in-the-wild exploitation. Chrome,
Adobe, and VMware shipped fixes for critical flaws in the same release wave.
Admins should prioritize the Firefox updates given the public exploit code.
