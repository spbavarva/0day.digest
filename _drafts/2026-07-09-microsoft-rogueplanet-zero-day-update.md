Follow-up on the "RoguePlanet" Windows Defender zero-day first reported in
June: researcher "Nightmare-Eclipse," who has disclosed several other
Microsoft zero-days, published a working proof-of-concept in early June
that let a local attacker spawn a SYSTEM-level command prompt via a race
condition in Defender. Dark Reading now reports Microsoft has moved to
contain the threat, though the source does not specify whether a patch has
shipped or what mitigation was applied.
