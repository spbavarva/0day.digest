Microsoft publicly advocated for Coordinated Vulnerability Disclosure (CVD) after
removing the GitHub account of researcher Chaotic Eclipse (aka Nightmare-Eclipse),
who had publicly dropped details of multiple unpatched Microsoft zero-days. Microsoft
argues vendors need time to understand impact and ship fixes before public disclosure.
The incident highlights the ongoing tension between independent researchers who
believe delayed disclosure lets vendors sit on bugs, and vendors who say premature
disclosure endangers users. Researchers considering disclosure timelines should note
that Microsoft is actively enforcing this position through platform-level account
action.
