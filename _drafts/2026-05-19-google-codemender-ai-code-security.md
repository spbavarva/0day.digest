At Google I/O, Google announced wider external access to CodeMender, an AI agent for code security originally debuted in October 2025.
Select groups of security experts are being invited to test the CodeMender API, positioning it as a direct competitor to Anthropic's Mythos in AI-assisted vulnerability detection.
The broader rollout signals Google's push to establish a commercial presence in AI-powered application security tooling alongside its existing security product portfolio.
