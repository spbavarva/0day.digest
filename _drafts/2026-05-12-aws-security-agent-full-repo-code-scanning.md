AWS released a preview of full repository code scanning in AWS Security Agent, adding deep
context-aware security analysis across entire codebases. The AI-driven feature can identify
vulnerabilities and build working exploits to validate them, going beyond point-in-time SAST
by reasoning across the full repository context rather than individual files.

The capability positions AWS as a direct competitor to third-party tools like Snyk, Semgrep,
and GitHub Advanced Security for cloud-native development teams. Available in preview now;
AWS customers can opt in through the Security Agent console.
