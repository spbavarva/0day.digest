RansomHouse has claimed responsibility for last week's breach of Trellix's source code repository,
publishing screenshots as proof of access to internal systems. Trellix is a major cybersecurity
vendor formed from the merger of McAfee Enterprise and FireEye's enterprise business. Source code
exposure from a security vendor carries elevated risk: detailed knowledge of product internals can
accelerate exploit development against customers. No patch advisories have been issued. Trellix
customers should monitor for vendor updates and heighten anomaly detection on endpoints running
Trellix products.
