Wiz disclosed a new attack method against AI coding assistants that it
calls GhostApproval, describing it as a decades-old technique revived
against modern AI coding tools. The technique tricks AI coding assistants
into executing attacker-controlled code on the developer's own machine.
Affected tools and mitigation guidance were not specified in the available
summary; check Wiz's original disclosure for details.
