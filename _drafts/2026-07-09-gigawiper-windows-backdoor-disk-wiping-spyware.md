Microsoft has analyzed a new destructive Windows backdoor it calls GigaWiper.
Rather than a single payload, it bundles three older destructive techniques
into one tool that operators can select from: a full disk wipe, overwriting
the Windows system drive, and a fake "ransomware" mode that encrypts files
with a key it never saves — meaning victims cannot recover data even by
paying. Microsoft frames it as three known destructive techniques repackaged
into a single operator toolkit rather than a wholly new capability.
