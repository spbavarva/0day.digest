A new Linux kernel vulnerability dubbed "DirtyClone," a variant of the
earlier DirtyFrag flaw, lets unprivileged local users manipulate the page
cache and escalate to root privileges. Exploitation requires local access to
the target system. No CVE identifier, affected kernel version range, or
patch status was included in the source summary; check upstream kernel
advisories for remediation guidance.
