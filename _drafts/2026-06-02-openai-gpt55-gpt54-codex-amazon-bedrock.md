OpenAI's GPT-5.5, GPT-5.4, and Codex (the OpenAI coding agent) are now generally
available on Amazon Bedrock with pay-per-token pricing and Bedrock's built-in security
and governance controls. AWS customers can now access OpenAI frontier models through
Bedrock without direct OpenAI API credentials or a separate billing relationship.

The availability on Bedrock means these models inherit Bedrock's VPC endpoints, AWS
IAM-based access control, and CloudTrail audit logging — relevant for regulated
industries running AI workloads under compliance requirements.
