Kaspersky's GReAT team detailed a global campaign distributing malicious VBScript files via WhatsApp that ultimately deploys a UEMS remote monitoring and management (RMM) agent through a multi-stage infection chain. Abusing legitimate RMM software gives attackers persistent remote access that blends in with normal IT tooling, making detection harder. Organizations should monitor for unexpected RMM agent installations and treat unsolicited script attachments in messaging apps as high-risk.
