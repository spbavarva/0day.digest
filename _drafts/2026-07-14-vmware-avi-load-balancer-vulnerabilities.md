VMware has patched seven vulnerabilities in its Avi Load Balancer product that can be exploited for authentication bypass, remote code execution, privilege escalation, and directory traversal. The load balancer is commonly deployed at the edge of enterprise application infrastructure, making unpatched instances an attractive target. No detail on active exploitation has been reported. Administrators should prioritize patching given the combination of RCE and auth-bypass primitives in the disclosed set.
