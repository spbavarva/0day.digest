Threat actors are actively exploiting a critical authentication bypass vulnerability in the
Burst Statistics WordPress plugin, gaining admin-level access to affected sites. The Burst
Statistics plugin is a traffic analytics tool used across a significant number of WordPress
installations.

WordPress site operators using Burst Statistics should update the plugin to the patched version
immediately or disable it until the update can be applied. Admin-level access via plugin exploits
is typically used to inject malicious code into site templates, install backdoors, or redirect
visitors to phishing infrastructure.
