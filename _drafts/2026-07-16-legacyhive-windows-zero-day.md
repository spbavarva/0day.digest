A researcher operating under the handle "Nightmare Eclipse" has published a
Windows zero-day vulnerability dubbed "LegacyHive." The researcher
intentionally stripped down the proof-of-concept exploit code to prevent
immediate exploitation of the flaw. No Microsoft patch is currently
available. Further technical details, including affected Windows versions
and the vulnerability class, were not disclosed in initial reporting.
Windows administrators should watch for an official Microsoft advisory and
apply mitigations once one is published.
