A solo Russian-speaking threat actor known as "bandcampro" used Google's
open-source Gemini CLI to help run parts of a botnet built from eight
compromised dental clinic PCs. Analysis of roughly 200 Gemini CLI session
logs from March 19 to April 21, 2026 showed the actor using the tool for
tasks including cracking passwords and setting up residential proxy
infrastructure. The botnet's scale is small, but the case is a concrete
example of an attacker operationalizing an open-source AI CLI tool for
offensive tasks.
