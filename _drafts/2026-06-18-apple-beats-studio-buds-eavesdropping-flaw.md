Apple released security updates patching a high-severity flaw in the
Beats Studio Buds wireless earbuds that could let an attacker within
Bluetooth range spy on a user's conversations. Apple has not detailed
whether the flaw was exploited in the wild. Beats Studio Buds owners
should install the latest firmware update as soon as it's available on
their device.
