More than 900 internet-exposed Oracle E-Business Suite (EBS) instances are
currently under active attack, targeting a critical flaw in the platform.
EBS handles core ERP functions — finance, HR, supply chain — for large
enterprises, making exposed instances a high-value target. No CVE identifier
or patch status was included in the available reporting. Organizations
running EBS should confirm exposure and check current Oracle security
advisories immediately.
