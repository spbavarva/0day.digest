IBM and Red Hat announced Project Lightwell, a $5 billion initiative to improve
security across open source software supply chains. The program is designed to
remediate vulnerabilities in widely deployed open source components without breaking
existing production workloads — addressing a core challenge in supply chain hardening.
The commitment comes as open source supply chain compromise remains a top attack
vector for both nation-state actors and ransomware operators. Further technical
details on the tooling and processes Lightwell will fund have not yet been published.
