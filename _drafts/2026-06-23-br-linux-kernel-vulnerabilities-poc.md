CISA warned that B&R industrial products ship with Linux kernel versions
affected by publicly known vulnerabilities that could allow an attacker to
escalate privileges locally. Public proof-of-concept exploits for these
kernel flaws are already available. B&R says it has no evidence of active
exploitation against its products at this time. Operators should check
vendor guidance for patched kernel versions for affected B&R equipment.
