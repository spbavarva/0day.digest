Attackers are actively exploiting a flaw in the Gravity SMTP WordPress
plugin to harvest sensitive data from vulnerable sites. Affected plugin
versions leak API keys, secrets, authentication tokens, and server
information to attackers. Site operators running Gravity SMTP should
update to a patched version immediately and rotate any credentials or
tokens that may have been exposed. The exact vulnerable version range
was not detailed in initial reporting.
