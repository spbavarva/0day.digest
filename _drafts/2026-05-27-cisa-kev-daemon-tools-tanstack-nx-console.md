CISA added three CVEs to its Known Exploited Vulnerabilities catalog:
CVE-2026-8398 (Daemon Tools Lite embedded malicious code), CVE-2026-45321
(TanStack unspecified vulnerability), and CVE-2026-48027 (Nx Console embedded
malicious code).

The Daemon Tools and Nx Console entries are supply chain compromises involving
malicious code embedded in software distributions — campaigns previously
documented in this digest. The KEV listing confirms active exploitation for all
three. Federal agencies must remediate under BOD 22-01 timelines; all other
organizations should verify these packages are patched or replaced in their
environments.
