Ubiquiti shipped updates addressing multiple critical flaws across UniFi
Connect, UniFi Talk, UniFi Access, UniFi Protect, and UniFi OS. The most
severe, CVE-2026-50746 (CVSS 10.0), is an improper access control
vulnerability in UniFi Connect Application that could allow privilege
escalation and arbitrary command execution. Admins running any of these
UniFi products should update immediately.
