Researchers have demonstrated MemGhost, an attack that abuses AI
assistants with persistent memory and inbox access. A single crafted
email can trick the agent into saving a false "fact" about the user,
hiding the change, and quietly steering its answers in later sessions.
Because the manipulation happens silently, the user sees an
ordinary-looking reply with no indication the assistant's memory was
tampered with. The attack underscores the risk of granting long-term
memory and autonomous email access to AI agents without safeguards
against injected content.
