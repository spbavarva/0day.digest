PaddlePaddle released PP-OCRv6 on Hugging Face, an OCR model family supporting 50 languages with variants ranging from 1.5M to 34.5M parameters. The size range targets deployment from lightweight edge devices up to higher-accuracy server use. No further technical detail was available in the source listing at publication time.
