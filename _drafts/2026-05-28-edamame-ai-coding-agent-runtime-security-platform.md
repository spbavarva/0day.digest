French startup Edamame has launched a runtime verification platform that uses host
telemetry and AI analysis to detect when coding agents deviate from their intended
behavior — termed "intent drift" — alongside secret theft and supply chain attacks
in real time. The tool targets the growing risk of autonomous AI coding agents
taking unintended or malicious actions inside development pipelines. As AI coding
agents gain broader enterprise adoption, runtime monitoring for agent behavior is
becoming a distinct security category. Security architects evaluating AI coding
tooling should assess what runtime visibility they have into agent actions.
