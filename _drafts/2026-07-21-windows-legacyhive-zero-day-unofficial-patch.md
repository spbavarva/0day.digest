A Windows zero-day dubbed "LegacyHive" allows local attackers to escalate
privileges on fully up-to-date Windows systems. Microsoft has not yet
released an official patch, but free unofficial patches are now available
from third-party sources. Organizations concerned about exposure should
evaluate the unofficial patch or apply compensating controls until Microsoft
ships an official fix.
