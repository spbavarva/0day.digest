Free unofficial patches are now available for LegacyHive, a recently
disclosed Windows zero-day flaw that allows attackers to escalate
privileges on fully up-to-date Windows systems. No official Microsoft fix
is available yet, leaving unofficial third-party patches as the only
mitigation in the meantime.
