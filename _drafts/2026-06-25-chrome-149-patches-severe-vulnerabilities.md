Google's Chrome 149 update resolves 18 vulnerabilities rated high or
critical severity. More than half are use-after-free defects, a bug class
that can potentially be chained into remote code execution in the
renderer or browser process. No active exploitation was reported for any
of the fixed issues. Users and IT teams should update Chrome to the latest
stable version given the volume and severity of memory-safety fixes in
this release.
