Microsoft announced two new LLMs at Build 2026: MAI-Thinking-1, a 1-trillion-parameter mixture-of-experts reasoning model with 35B active parameters, currently available to select early partners; and MAI-Code-1-Flash, a 137B-parameter MoE code model (5B active) purpose-built for GitHub Copilot, already rolling out to Copilot individual users in VS Code. Both models use sparse MoE architectures to reduce inference cost relative to parameter count. This marks Microsoft's first publicly disclosed in-house LLM series developed independently of its OpenAI partnership, signaling a strategic shift toward internal model capacity.
