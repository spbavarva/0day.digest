AWS introduced the Amazon GuardDuty investigation agent, now in public
preview, an AI-powered feature that automatically investigates security
findings across AWS accounts and workloads. AWS says the agent reduces
investigation time from hours to minutes by correlating findings and
context automatically. This extends GuardDuty's managed threat detection
with an agentic triage layer for security teams.
