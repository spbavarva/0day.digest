AWS added AI workload protection and multicloud support for Microsoft Azure
to Security Hub, its centralized cloud security operations service. The AI
workload protection capability is aimed at giving teams visibility into
risks specific to AI workloads running in AWS environments.

The Azure support extends Security Hub's centralized findings and
prioritization across a second major cloud provider.
