Anthropic has expanded Claude's enterprise security governance capabilities with 28 new integrations spanning major security vendors: CrowdStrike, Palo Alto Networks, Microsoft, Okta, Zscaler, Netskope, Cloudflare, Fortinet, and Wiz among them.

The integrations target governance, investigation, and compliance use cases, positioning Claude as a cross-platform AI layer in enterprise security stacks. Organizations evaluating AI-assisted security operations should assess how Claude's data access model aligns with existing DLP policies before enabling integrations with identity and network security tooling.
