Microsoft has open-sourced two AI agent security testing tools: RAMPART (Risk Assessment and Measurement Platform for Agentic Red Teaming), a Pytest-native framework for writing and running safety and security tests for AI agents, and Clarity, which helps developers assess agent security posture during development.

Both tools target the gap between how quickly AI agents are deployed and how rarely they receive purpose-built security testing. Teams building or deploying AI agents can use RAMPART to write structured safety test suites alongside functional tests, reducing the chance that security properties are only evaluated after incidents occur.
