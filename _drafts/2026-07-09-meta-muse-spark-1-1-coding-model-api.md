Meta has opened its Muse Spark 1.1 model to third-party developers via a
new Meta Model API, following the original Muse Spark's April debut. The
company says the update is a "step-change" from the first generation, with
improvements aimed at agentic tool calling, computer use, and coding
tasks. This marks Meta's first Muse Spark model to be exposed via a public
API rather than only through Meta's own products.
