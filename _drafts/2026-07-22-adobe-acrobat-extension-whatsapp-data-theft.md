A flaw in the Adobe Acrobat extension for Chrome, installed on more than
300 million browsers, let any website read conversations and data rendered
in WhatsApp Web without any authentication. An attacker only needed to
convince a targeted user to visit a malicious site to exfiltrate WhatsApp
messages and contacts. Users of the extension should update to the patched
version as soon as it's available.
