### 2. 148 npm Packages Disguised as Student Proxies Turned Browsers Into a DDoS Botnet
- **Source:** The Hacker News — https://thehackernews.com/2026/07/148-npm-packages-disguised-as-student.html
- **Severity:** high
- **Tags:** `supply-chain`, `npm`, `malware`
- **Summary:** JFrog found 148 npm packages disguised as student web proxies, used as free hosting for a booby-trapped proxy site that turned visitors' browsers into a DDoS botnet for roughly two weeks in May. Developers installing packages were not the target — students seeking proxy access were.
