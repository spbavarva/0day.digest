OpenAI has launched "Trusted Access for Cyber," a program that gives leading security firms
and enterprises access to GPT-5.4-Cyber alongside $10 million in API grants to accelerate
cyber defense tooling. The program is framed as a coordinated push to build AI-native
security capabilities across the industry.

GPT-5.4-Cyber appears to be a specialized model variant optimized for security use cases.
The $10M grant pool signals OpenAI is making a strategic play for the enterprise security
market in direct competition with Anthropic's Mythos and Google's security-focused AI
programs.

Organizations working on AI-assisted threat detection, vulnerability analysis, or security
automation may be eligible for grant access; details are available on the OpenAI blog.
