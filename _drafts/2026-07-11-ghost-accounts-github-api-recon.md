Multiple campaigns are using disposable "ghost" GitHub accounts to enumerate organizations via the GitHub API, mapping their repositories and member lists at scale. The activity appears to be reconnaissance rather than direct exploitation, likely building target profiles for follow-on attacks. Further technical detail wasn't available in current reporting.
