North Korean threat actors linked to the Contagious Interview campaign are hiding malicious payloads inside SVG image files using steganography, delivered through fake job postings and coding challenges aimed at developers. Anyone who ran the malicious project received a four-stage payload aligned with OTTERCOOKIE: a browser credential and crypto wallet stealer, plus a file stealer. Developers should treat unsolicited coding-test repos and take-home projects as untrusted code and run them only in isolated, disposable environments.
