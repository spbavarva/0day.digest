The Windows release of Hola Browser has been compromised in a supply chain attack that
bundled an undisclosed executable identified by researchers as a cryptocurrency miner.
Users who downloaded or updated Hola Browser for Windows during the affected window may
have cryptomining code running on their systems. Affected users should remove Hola
Browser, scan for cryptominer processes, and investigate unexpected CPU utilization.
