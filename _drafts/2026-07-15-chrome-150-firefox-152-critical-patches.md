Google and Mozilla shipped Chrome 150 and Firefox 152 with fixes for multiple
critical-severity vulnerabilities. Public exploit code exists for the patched
Firefox flaws, though no in-the-wild exploitation has been observed yet.

Administrators and users should update both browsers promptly given the
availability of public proof-of-concept code.
