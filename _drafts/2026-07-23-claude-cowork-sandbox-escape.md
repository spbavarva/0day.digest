Researchers at Accomplish AI disclosed a sandbox escape vulnerability in
Anthropic's Claude Cowork that could let the AI agent break out of its Linux
VM and read or write files anywhere on the host Mac. The issue was shared
with The Hacker News ahead of publication, indicating coordinated
disclosure. Roughly 500,000 macOS users running the affected Claude Cowork
configuration were reportedly exposed. Users should confirm they're on a
patched build once one is available, if not already.
