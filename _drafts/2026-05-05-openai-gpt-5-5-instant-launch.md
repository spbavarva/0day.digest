OpenAI has replaced ChatGPT's default model with GPT-5.5 Instant, claiming it delivers "52.5% fewer hallucinated claims" than its predecessor based on internal evaluations. The model targets improved factuality specifically in sensitive domains: law, medicine, and finance. OpenAI also published a system card alongside the release. The update preserves the low-latency characteristics of the previous default while adding improved personalization controls for users. GPT-5.5 Instant is now the default for all ChatGPT users; API access details were not specified in the announcement.
