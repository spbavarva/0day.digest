A critical security vulnerability in WP Maps Pro, a WordPress plugin with over 15,000 sales on Envato Market, is being actively exploited to create unauthorized administrator accounts on vulnerable sites. The plugin provides customizable Google Maps and OpenStreetMap integration with markers and location features. WordPress site owners running WP Maps Pro should disable the plugin immediately until a patched version is available. Administrators should audit the WordPress users table for recently added admin accounts and revoke any unauthorized access — successful exploitation grants full site control.
