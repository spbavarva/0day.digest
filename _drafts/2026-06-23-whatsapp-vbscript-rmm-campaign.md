An active campaign is distributing malicious VBScript files via WhatsApp
direct messages disguised as documents, leading to the installation of
legitimate Remote Monitoring and Management (RMM) software for malicious use.
Per Kaspersky, the campaign targets WhatsApp Desktop and WhatsApp Web users
across Malaysia, Brazil, India, Mexico, Singapore, the U.K., Spain, Taiwan,
and Australia. Users should be cautious of unexpected document attachments
sent via WhatsApp, particularly ones prompting them to run scripts.
