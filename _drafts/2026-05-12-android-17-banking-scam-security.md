Android 17, expected to roll out next month, will introduce enhanced security features
targeting banking scam calls, device theft, and user privacy. The update expands on
previous Android defenses against SIM-swap fraud and phone-based social engineering used
to compromise banking accounts. New controls include improved real-time scam call detection
and privacy tools that limit data accessible during suspicious call scenarios. Developers
building banking or financial applications for Android should review the new API surfaces
for scam-detection integration ahead of the release.
