Ubiquiti released security updates patching three maximum-severity vulnerabilities in UniFi OS, all exploitable remotely by unauthenticated attackers. CVE identifiers and technical details have not been disclosed. No active exploitation has been reported. Organizations running UniFi controllers, gateways, or access points should apply the available updates immediately given the unauthenticated remote attack surface and maximum severity rating.
