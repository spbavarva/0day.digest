A vulnerability dubbed "Rogue Agent" in Google Dialogflow CX could have
let attackers silently manipulate AI conversations, exfiltrate data, and
compromise every Dialogflow CX agent within the same Google Cloud
project. The issue has been addressed by Google. Organizations using
Dialogflow CX should confirm they are on a patched configuration and
review agent activity logs for anomalies.
