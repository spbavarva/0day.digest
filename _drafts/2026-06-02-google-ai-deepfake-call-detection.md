Google is rolling out a feature in its Phone app that detects calls where a scammer spoofs a number already in the user's contact list and uses AI-generated voice to impersonate a trusted contact, employer, or authority figure. When the system detects a mismatch between the incoming caller ID and expected voice patterns, it flags the call as suspicious so the user can hang up. The feature targets an emerging class of vishing attacks enabled by accessible real-time voice cloning tools. It represents one of the first consumer-level defenses explicitly designed to counter AI deepfake phone fraud.
