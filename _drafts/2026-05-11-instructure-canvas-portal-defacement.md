Instructure has confirmed that attackers exploited a security vulnerability in Canvas LMS to
modify login portal pages across customer institutions, replacing them with extortion messages.
The attack disrupted access for tens of thousands of students globally during final exam season.
Canvas has since restored service.

The specific flaw exploited has not been fully disclosed. The presence of an extortion message
suggests the attacker may have had leverage beyond simple defacement — potentially data access
or persistent access to affected tenants. Educational institutions using Canvas should review
admin and API access logs for the incident window. A full post-incident disclosure from
Instructure is pending.
