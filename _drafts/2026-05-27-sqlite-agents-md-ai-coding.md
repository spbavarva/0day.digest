SQLite added an AGENTS.md file to its codebase establishing explicit policies for AI coding agents: agentic code contributions are barred, but agents may submit bug reports provided a human reviews and submits them. Simon Willison flagged this as a notable early example of a major open-source project proactively defining AI agent governance. The policy distinguishes between agentic code (rejected outright) and agent-assisted bug discovery (permitted with human oversight) — a nuanced stance as AI coding agents become routine development tools. The pattern of AGENTS.md files is likely to spread across high-profile repositories.
