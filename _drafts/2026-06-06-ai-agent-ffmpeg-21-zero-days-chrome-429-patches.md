A security startup disclosed 21 previously unknown vulnerabilities in FFmpeg —
the media-processing library embedded in virtually every application that handles
video — all found autonomously by an AI agent. In the same week, Google released
Chrome 149 with patches for a record 429 security bugs; the FFmpeg findings were
AI-discovered, the Chrome bugs were not.

FFmpeg ships in browsers, streaming platforms, video editors, and embedded systems
globally, making the attack surface broad. Organizations should prioritize updating
bundled FFmpeg versions once patches are available. The demonstration signals that
autonomous AI-driven vulnerability research is maturing into a practical threat
surface expansion mechanism.
