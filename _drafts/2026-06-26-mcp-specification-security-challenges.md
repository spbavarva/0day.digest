A major overhaul of the Model Context Protocol (MCP) specification shifts
several critical security responsibilities away from the protocol itself
and onto developers and platform operators. The new enterprise-ready
specification changes how MCP previously governed certain security areas
more directly, meaning teams building or deploying MCP servers will need to
take on more of that security burden themselves.
