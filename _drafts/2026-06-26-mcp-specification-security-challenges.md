A major overhaul of the Model Context Protocol shifts critical security
responsibilities away from the protocol itself and onto developers and
platform operators integrating it. As MCP adoption grows for connecting
LLM agents to tools and data sources, teams building or deploying MCP
servers need to take on authentication, authorization, and input validation
work that the spec no longer handles for them.
