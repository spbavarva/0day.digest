A single attacker used AI workflows combined with chained cloud
misconfigurations and stolen credentials to breach a large Amazon
customer's AWS environment within 72 hours, then used the access to
extort the victim. The case illustrates how AI tooling can compress the
time attackers need to chain together cloud weaknesses that would
otherwise require more manual reconnaissance.
