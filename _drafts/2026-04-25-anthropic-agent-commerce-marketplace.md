Anthropic ran an internal experiment where AI agents acted as both buyers and sellers in a
classified marketplace, completing real transactions for real goods with real money. The test
explores whether autonomous agents can negotiate, agree on terms, and execute commerce without
human intermediaries.

Agent-on-agent markets introduce novel security concerns: spoofed agent identity, adversarial
negotiation tactics, financial loss from misconfigured agents, and the absence of human-in-the-loop
controls for transaction approval. No production deployment or public API surface was announced.
This remains an internal research experiment; practitioners building agentic systems should note
Anthropic is actively stress-testing autonomous financial decision-making at small scale.
