ESET researchers identified a new Android spyware campaign, codenamed Asin, that has targeted
Arabic-speaking users since early 2025 across multiple attack waves. The malware is distributed via
fake websites mimicking news utilities, war-related updates, and government news sources — including
the domain govlens[.]net. Each campaign wave uses distinct infrastructure and app lures to evade
detection. Users should install applications only from official app stores, verify developer identities
before installation, and use mobile security tooling capable of detecting sideloaded APKs.
