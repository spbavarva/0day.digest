OpenAI has released ChatGPT Images 2.0, the latest version of its AI-powered image generation
model. New capabilities include "thinking" features that allow the model to pull live information
from the web before generating images, improved instruction-following, and the ability to produce
multiple images from a single prompt.

The web-search integration marks a new interaction pattern for image generation — content can now
incorporate current real-world information, which has implications for content authenticity
verification and potential misinformation use cases. Security and trust-and-safety teams should
factor this capability into their AI-generated content detection and media verification workflows.
